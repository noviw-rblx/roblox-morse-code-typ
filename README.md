# Roblox Morse Code Communication Script

A Roblox script that allows players to communicate using character movements based on Morse code. Messages are automatically converted into movement patterns that can be understood by anyone familiar with the system and its does it automatically for you.

## Features

* Full Morse code alphabet support (A-Z)
* Automatic text-to-Morse conversion

## Language Used

* Lua (Luau)

## Controls

| Action       | Meaning                   |
| ------------ | ------------------------- |
| ● (Dot)      | Jump once                 |
| ‒ (Dash)     | Walk one step to the side |
| Letter Space | Stop for 0.7 seconds      |
| Word Space   | Stop for 1 second         |
